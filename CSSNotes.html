The Basics

  • To include CSS in your HTML document via a stylesheet, use a <link> element with an href attribute pointing to your stylesheet.
    For example:
    • <link rel="stylesheet" href="filename or URL">
    You can include CSS directly in your page using the <style> tag. Be mindful of the fact that making changes in the future will be difficult.

  • A CSS selector and the properties defined between its brackets is called, altogether, a rule.

  • Elements nested within others will inherit rules from their parent element.

  • To override an inherited rule, create a new rule for the element(s) to be overridden.
    • The most specific rule always wins in this case.

  • Style a specific element by creating a class selector. State the element type followed by a dot, followed by the class name. Then you can define rules to your heart's content!
    For example:
    • li.yellow{background-color:yellow;}
    And if you'd like to style ALL elements with the yellow class, simply omit the element name(s). Your syntax will then be .class
    • .yellow{background-color:yellow;}

  • Elements can belong to more than one class as well. List all the classes between quotes followed by spaces.
    For example:
    • "yellow centered_text important"

  • These rules range in specificity. Element and class themselves equally are less specific than "element.class".
    • The most specific selector wins in terms of rule conflict, such as when rules format the same property differently.
    • If even this fails, the lower the rule is on the stylesheet, the more superiority it has. This list element belongs to the yellow and orange classes, which both only designate background color. The orange rule is lower on the page than the yellow rule.

  • ID's can be selected by using a # symbol instead of a dot. Remember to give any single element one unique ID.
    • #id-name
  • You can even override stylesheets! The lower they're <link>'ed in the head element, the more precedence they have over previous sheet(s).

  • Use "/* */" to comment in CSS.
    For example:
    • /* This is a comment. */


Fonts

  • A font family refers to either:
    • A group of fonts that share design characteristics, i.e. sans serif, serif, monospace, fantasy...
    • A set of font variations based off one parent font, i.e. gothic, gothic bold, gothic italic...

  • To include a custom font, either from the web or from your own server, use the font face rule.
    For example:
    • @font-face {
      font-family: "Name of your Font";
      src: url("relative or absolute url for font of one file type"),
      src: url("another url for font of another file type for compatability"); }

  • Some important font properties to know:
    • font-size: 14px, 15%, 1.5em, medium;
    • line-height: 13px, 95%, 1.9em;
    • font-weight: bold, normal;
    • font-style: italic, oblique;
    • text-decoration: underline, overline, line-through
    em and % are proportional modifiers.


The Box Model

  • CSS treats HTML body elements like "layered" boxes. This is referred to as the Box Model.


    This box consists of four components:
    • Content
    • Padding
    • Border
    • Margin
    The content area contains, of course, the content contained in the element.
    • width: 600px;
    The padding box provides space between the content and the border.
    • padding: 25px;
    • padding-left: 80%;
    Borders contain the content and padding within a box whose appearance can be customized.
    • border: solid 4px #c45525;
    • border-style: solid, groove, dotted;
    • border-width:thin, 9px;
    • border-color: red, rgb(26%, 12%, 75%), #ff0011;
    The margin box provides space between the border and other elements.
    Adjust padding and margin sizes with:
    • margin: 30%;
    • margin-right: 50px;
    Add a position between "border" and property to edit specific borders.
    For rounded borders:
    • border-top-left-radius: 3em;
    • border-bottom-right-radius: 0px;
    The dimensions of an element are the sums of the widths and lengths of their boxes, and thus their components thereof.
    It's also recommended not to set the height of your element, lest its contents spill over into other elements under unforeseen browser circumstances.

  • Add a decorative image to an element using the background image property.
    For example:
    • background-image: url(url_goes_here.jpg);
    Use this only when an image does not add function to your page, and is used only for decorative purposes.
    Images used this way, by default, are generated in the top left of the element and repeat in a tiled manner.
    Address the repeating and positioning of your background image with these properties:
    • background-repeat: no-repeat, repeat-x, inherit;
    • background-position: left, bottom, center, 12px, 43%;
