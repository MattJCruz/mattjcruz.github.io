The Basics

I will not be including basic programming definitions, i.e. variables, booleans, if statements etc.
  • Javascript is a scripting language, a programming language that's interpreted or compiled at runtime, as opposed to being compiled before runtime as is usually the case with Object Oriented Programs.

  • Declare variables with the var keyword.
  • Name variables starting with a letter, underscore, or dollar sign, and every subsequent character can be one of the above or a number.
    • Additionally they're case-sensitive. NUM1 and num1 are two different variable names.

  • Application Programming Interfaces (API's) are used to access additional functionality beyond the scope of a simple program.

  • End statments with a semicolon.

  • Strings are surrounded with double quotes.

  • A variable with a set value will be considered true in a boolean expression, while an empty one will be considered false.

  • Use console.log to write to the javascript console in your environment.

  • To write to an HTML document in Javascript, use the document.write function. This is not recommended as there are (apparently?) better ways to do this.
