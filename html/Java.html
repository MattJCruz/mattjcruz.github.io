Under the Hood

Java is a platform-independent programming language. This means you can write a program once and it'll run anywhere (you can run java)!

Running a Java Program

  1. A java file consists, broadly, of three components: Classes, Methods, and Statements.
  2. Write your java code in the source document, a.k.a the source file.
  3. Compile your code to ensure it will run without forseeable errors. This turns it into Java bytecode.
  4. Run the compiled Java bytecode on the Java Virtual Machine (JVM), the software installed on devices which used to run Java code.

Anatomy of a Java Program


This is what goes inside:

    Methods

  • Methods are the programming in computer programming. They contain statements directed toward fulfilling a particular function. They're reusable packets of code

    • Classes

  • It's standard practice to only include one class per source file, as the file is named after the class contained within. Classes are referenced by file name so having multiple classes oin one source file would be counterproductive.

    • Statements, Loops, & Branches

In statments such as x=x+3, the expression is evaluated before assignment.
